# Mood_Videos

A simple web app that helps you pick videos based on your current mood.

## Overview

You select a mood.  
The system returns a list of matching videos.

The goal is fast selection. No complex UI. No login. No storage.

## Features

- Choose mood (happy, sad, focus, relax, etc.)
- Get instant video suggestions
- Lightweight and fast
- Easy to customize video list

## Use Cases

* Quick mood-based entertainment
* Focus music selection
* Relaxation support
